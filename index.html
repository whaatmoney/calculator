Reference

Quick reference — decision steps
  1. Enter dimensions (L, W, H in inches).Volume and Base Price auto-calculate.
  2. Add hard cost adders.Cavity, Spec Fee via End User, and Packaging. Length Surcharge auto-calculates from longest dimension.
  3. Select Process Factor.Passivation / VC / 300 / 200 / 100 / 50 / 25. Passivation = 1.00 — not a cleanliness level.
  4. Verify Size Factor.Auto from longest dimension; override if needed. Small ≤6", Medium 6–12", Large 12–24", XL >24".
  5. Select Weight Factor.<10 lb / 10–40 lb / 40–100 lb / >100 lb.
  6. Select Complexity.Replaces legacy geometry multiplier. Standard / Multi-port / Manifold / Aerospace critical.
  7. Read Unit Price.Check flags for guardrails before sending the number.

Calculator policy
  • Cleanliness or process is selected first. Passivation only uses a factor of 1.00.
  • Size, weight, and complexity are handling/process multipliers.
  • Cavity, length, spec, and packaging remain additive.
  • Complexity replaces legacy geometry multiplier to prevent double-counting.
  • Do NOT stack old geometry multiplier with new complexity multiplier.
  • If Aerospace critical is selected, an estimator note is required.
